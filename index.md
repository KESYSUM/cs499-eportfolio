## Professional Self-Assessment

Completing the Computer Science program at Southern New Hampshire University has helped me develop both technical and professional skills that prepare me for a career in software development. Throughout the program, I gained experience with software engineering, databases, algorithms and data structures, security, testing, and mobile application development. Creating this ePortfolio allowed me to reflect on how much I have grown as a developer and how the knowledge from different courses connects together when building software solutions.

One important skill I developed throughout the program was communication. Whether writing project documentation, discussing design decisions, or completing code reviews, I learned how to explain technical concepts clearly to different audiences. Collaboration was also an important part of the program because software development often requires working with others, accepting feedback, and making improvements based on that feedback. The capstone code review process reinforced the importance of evaluating software and communicating enhancement plans in a professional manner.

The program significantly improved my software engineering abilities. Through coursework and projects, I learned how to design maintainable applications, organize code effectively, and follow development best practices. My capstone artifact, an Android inventory management application, demonstrates these skills through enhancements that improved validation, organization, security, and overall functionality. I also expanded my understanding of algorithms and data structures by implementing searching, sorting, and filtering features that improved how inventory records are managed and displayed within the application.

Database management and security became some of the most valuable skills I developed throughout the program. Working with SQLite and other database technologies taught me the importance of data integrity, validation, and efficient data management. Security concepts were reinforced through several courses and were applied directly within my capstone project by implementing salted SHA-256 password hashing, stronger authentication logic, and improved validation controls. These enhancements helped protect user information and demonstrated the importance of secure software development practices.

The artifacts included in this portfolio work together to demonstrate my growth throughout the Computer Science program. The software engineering enhancement highlights improvements to application design and maintainability. The algorithms and data structures enhancement demonstrates improved efficiency through searching, sorting, and filtering functionality. The database enhancement focuses on secure data management and database reliability. Together, these artifacts showcase the skills and knowledge I have developed throughout the program and demonstrate my readiness to continue growing as a software developer.

Overall, the Computer Science program has provided me with a strong foundation in software development, problem solving, database management, algorithms, and security. The knowledge and experience gained throughout the program have prepared me to enter the field with confidence and continue developing my skills as a professional software developer.
